# Wikipedia do calouro
## Sobre o projeto
este projeto tem como objetivo criar uma "Wikipedia do calouro", um guia simples para ajudar novos alunos que estão iniciando o curso. A ideia é reunir informações importantes sobre as disciplinas, além de algumas dicas úteis para facilitar a adaptação ao curso.
## Estrutura do repositório
o repositório está organizado nos seguintes arquivos: 

- **disciplinas.md** - Contém uma descrição das principais disciplinas do curso.
- **dicas.md** - Apresenta algumas dicas e recomendações para os novos alunos.
- **lanchonetes.md** - Este guia apresenta lugares de qualidade para se alimentar, com preços e descrições de cada local.
